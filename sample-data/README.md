**How to use**

1. Create workspaces for any month (Sidebar → Add a month / New workspace), e.g. **Jan 2026** … **Dec 2026**.
2. Select a month → Dashboard → Upload the matching CSV (e.g. **May 2026** → `may-2026.csv`).
3. Switch between **Overview** and each month to see totals, subscriptions, price creep, categories, vendor analytics, negotiation candidates, and cancellation queue all update per view.
